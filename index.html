Markdown Studio Guide

Everything you need to master your premium workspace.

1. First-Time Setup & Syncing

By default, Studio runs in Local Storage (your data never leaves your browser). To enable cross-device syncing, you can connect your GitHub account.

  • Click Cloud in the top navbar.
  • You will be prompted for a GitHub Personal Access Token (PAT).
  • Click "Need Help?" to see instructions on generating a token. Ensure you check both repo and gist scopes.
  • Once connected, all your notes and folders are securely synced to a private repository named markdown-studio-notes on your GitHub.

2. Custom Pro Syntax & Shortcuts

Studio extends standard Markdown with powerful custom features tailored for writers and designers.

🎨 Text Colors

Wrap text in brackets and specify a color inside braces.

[This is red text]{red} [Custom Hex Color]{#10b981} [Adaptive White/Black]{white} -> Changes based on Dark/Light theme automatically!

🖼️ Advanced Image Control

Control the size and alignment of images instantly.

![Alt](image_url.jpg){300x400, center} -> Width 300px, Height 400px (Center) ![Alt](image_url.jpg){500, left} -> Width 500px, Auto Height, floats left ![Alt](image_url.jpg){100%, center} -> Full responsive width

↔️ Text Alignment

Use slash commands at the start of a line or to wrap blocks.

/center This text will be centered /right This text aligns right

➖ Premium Dividers

Type three equals signs on a new line to generate a double-dashed stylish separator.

===

📐 Raw SVG Injection

Unlike standard editors, you can paste raw scalable vector graphics directly into the editor. They will render beautifully without security blockage.

<svg xmlns="...">...</svg>

3. Features & Buttons

  • 📂 Folder System: Organize notes into folders. On Cloud mode, these map directly to sub-directories in your GitHub repo.
  • 🖨️ PDF Export: Highly optimized print engine. Choose between A4 (Standard), A2 (Large), or Infinity (Single continuous page). Text auto-adapts color based on theme.
  • 🔒 Secure Share: Generates a highly secure link using AES-256 encryption stored in a Secret Gist. The decryption key is embedded in the URL hash (never sent to the server).
  • 📄 Import/Export: Click the ⬇️ button to download your raw .md file, or ⬆️ to import any text/markdown file locally.

4. Hidden Features (Pro Tips)

  • Zero-Lag Engine: Studio uses a custom 400ms Debounce Engine. You can write a 300,000+ word book and the typing will remain 100% smooth without blocking the main thread.
  • Code Copy: Hover over any rendered code block in the preview pane to reveal a sleek "Copy" button.
  • Draggable Pane: On Desktop, drag the vertical line between the editor and preview to resize your workspace.
  • Tab Indentation: Pressing Tab in the editor inserts actual spaces instead of trapping focus.
  • Failsafe Save: Closing the tab accidentally mid-typing will trigger an emergency state-save.

Link your account to sync notes to the cloud.

How to get a PAT?

Follow these steps to generate a secure token for syncing and sharing.

  • 1
    Go to GitHub Settings > Developer settings.
  • 2
    Click on Personal access tokens → Tokens (classic).
  • 3
    Click Generate new token (classic).
  • 4
    Give it a name, and strictly check BOTH the repo AND gist checkboxes.
  • 5
    Click Generate token, copy it, and paste it here!
